# 11. 환경설정 메뉴 · 난이도/중요도 · 형광펜 · 인라인 마크다운 (2026-07-17)

## 대표 지시

1. 선수학습 다음에 "환경설정" 메뉴
2. 모든 내용에 난이도와 중요도 표기
3. 형광펜 표기
4. 마크다운(`**단어**` 등)을 정확히 HTML(볼드 등)로 반영

## 구현

### 인라인 마크다운 렌더러 (utils/md.jsx)
- `**텍스트**`→<strong> / `==텍스트==`→<mark class="hl"> 형광펜 / 백틱→<code>
- 산문 필드 렌더링 지점에 전부 연결: 교시 이론·학습목표, 선수학습, 환경설정, 프로젝트, 퀴즈 해설
- **주의 2가지**: ① code.source에는 적용하지 않음(파이썬 `**` 연산자 오인) ② ==...== 안에 **...**를 중첩하면 파싱 안 됨(1단계 파서) — 중첩 금지
- **데이터 파일은 JS 템플릿 리터럴이라 산문에 백틱(\`) 문법을 쓰면 문자열이 깨짐** — 실제로 day3에서 빌드 깨진 사고 있었음. 산문에서는 볼드/형광펜만 사용할 것
- 형광펜 스타일: 노란 아래쪽 하이라이트(mark.hl), 다크 테마는 투명도 낮춤

### 난이도·중요도 (1~5 별)
- 21교시 전체에 difficulty/importance 필드 삽입 (add_levels.mjs — kind: 줄 뒤 자동 삽입)
  - Day1은 대체로 난이도 2~3, Day3 후반 SAC/TAC는 5. 중요도는 벨만·TD·DQN·PG·SAC 핵심 교시가 5
- 표기 위치: 교시 페이지 헤더 + 일차 목록 행 (노랑=난이도, 오렌지=중요도 별)
- 선수학습 4영역 difficulty 추가(level=필수/권장이 중요도 역할), 프로젝트 8종 difficulty/importance 추가
- stars() 헬퍼: ★★★☆☆ 형식

### 환경설정 메뉴 (/setup, 선수학습 다음)
- 선수학습 5번 "실습 환경 준비"를 독립 페이지로 승격·확장 (선수학습은 4영역으로 축소 + /setup 안내)
- 5단계: 파이썬&아나콘다(conda 가상환경) / 필수 라이브러리(+GPU CUDA 셀렉터) / 설치 확인 / Colab(.py import 팁) / VS Code 권장 설정(한국어팩·Python 확장·밝은 테마)
- 각 단계 난이도·중요도 별점 + 책갈피 사이드바 + 바로가기 카드

### 형광펜·볼드 콘텐츠 적용
- 21교시 이론에 68곳 마크업: 각 교시 핵심 문장 ==형광펜== + 핵심 용어 **볼드** (스크립트 일괄, [MISS] 0)
