# 20. 강의 도해(SVG) 13종 · 스케이트 일화 (2026-07-18)

## 대표 지시

1. SVG/그래프 설명이 필요한 부분에 이미지·그래프 포함
2. 강화학습 도입에 대표의 어린 시절 일화 사용 — 스케이트를 아버지께 배울 때 본인은 손 잡고 정석으로, 남동생은 넘어지며 스스로 배웠고 결국 남동생이 더 잘 탔다는 이야기 (자전거 비유도 함께)

## 구조

- `components/figures.jsx` — 도해 라이브러리. FIGURES[id] = { caption, C(컴포넌트) }
- 이론 블록에 `fig: '<id>'` 필드 → SessionPage가 본문 아래 <Figure> 렌더링
- 색은 전부 CSS 변수 → 라이트/다크 자동 대응, 곡선·막대 좌표는 JS로 계산(softmax·γ^t 등 실제 수식)
- marker(화살촉) id는 도해별 접두사 — 한 페이지 다중 도해 충돌 방지
