# 13. 기준 가로폭 확대 · 실습소스 일자별 탭 (2026-07-17)

## 기준 가로폭 (대표 질문: "1450 기준인가?")

- 기존 기준은 **1080px**이었음 (.container max-width) — 1450 아님
- 사이드바(190px) 도입으로 본문이 ~860px까지 좁아져 **1280px로 확대** (본문 약 1040px 확보)
- 이론 텍스트 가독성을 위해 1400+ 대신 1280 선택

## 실습소스 일자별 탭

- 3일치 코드 세로 나열 → **DAY 1/2/3 탭 전환**으로 변경 (탭에 일차 테마·코드 개수 표시)
- 사이드바 책갈피는 현재 탭 일차의 교시만 표시
- 탭 CSS(.tab-row/.tab)는 공용 — 다른 페이지에서 재사용 가능
