# 03. 라이트/다크 모드 (2026-07-17)

## 배경

대표 피드백: 어두운 배경이라 모드 조정 필요 → 강의장 빔프로젝터 환경을 기본으로 **라이트 모드 기본** + 헤더 토글(🌙/☀️) 추가.

## 구현

- `:root[data-theme='light'|'dark']` CSS 변수 2벌, `<html data-theme>`로 전환
- index.html 인라인 스크립트가 첫 페인트 전에 `localStorage('pytorch26_theme')` 적용 (깜빡임 방지, 기본 light)
- Layout.jsx 토글 버튼 — 상태는 localStorage 영속
- **코드 블록(`--code-bg`)은 두 모드 모두 다크 고정** — 파이썬 코드 가독성 때문. 라이트 모드로 바꾸지 말 것
- 인라인 code 색을 하드코딩(#ffa657) → var(--accent)로 교체 (라이트에서 대비 확보)
