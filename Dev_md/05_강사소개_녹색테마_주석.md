- `--accent`: light `#15803d` / dark `#3fb950` (테마별 별도 정의 — 다크는 밝게)
- 파비콘 SVG도 녹색으로 교체

## 3) 상세 주석 (인수인계용)

전 소스에 파일 헤더 주석 추가 — 세션·모델이 바뀌어도 이어갈 수 있도록 "왜"를 포함:

- main.jsx: HashRouter를 쓰는 이유(BrowserRouter 금지 사유)
- App.jsx: 전체 라우트 맵
- Layout.jsx: 테마 전환 3단계 동작(진실의 원본 = html data-theme), THEME_KEY는 index.html과 동일해야 함
- CodeBlock.jsx: props 스키마, 하이라이터 없는 이유, clipboard는 https 전용
- useProgress.js: localStorage 스키마, 서버 저장으로 바꿀 때 이 훅만 교체하면 되는 구조
- curriculum.js: 세션 데이터 스키마 전체, 교시 추가 방법
- day1~3.js: 교시 간 코드 재사용 흐름(GridWorld→DP→TD, ReplayBuffer→DQN→DDPG→SAC→TAC)
- quizzes/extrastudy/prereq.js: 데이터 스키마
- global.css: 컬러 시스템 규칙(code-bg 다크 고정, --red 분리 사유)
- 각 페이지: 화면 구성·특이 동작(findAdjacent 일차 경계 처리, 마지막 교시 안내 박스 등)
