# 06. 컬러 정정 — 메인 오렌지 복원 + 코드 주석만 녹색 (2026-07-17)

## 배경

05에서 시그니처 컬러를 녹색으로 바꿨으나 이는 지시 오해.
**대표 확정안: 메인 컬러 = PyTorch 오렌지(#EE4C2C) 유지, 녹색은 실습 코드 블록 안의 파이썬 주석(#...) 폰트에만 적용.**

## 변경 사항

1. `--accent`를 두 테마 모두 `#ee4c2c`로 복원, 파비콘도 오렌지 복원
2. `--red`(퀴즈 오답 전용)는 유지 — 오답을 accent로 칠하던 초기 구조보다 명확해서 존치
3. CodeBlock.jsx에 경량 주석 하이라이팅 추가:
   - `findCommentStart()`: 따옴표 상태를 추적해 "문자열 밖의 첫 #"만 주석으로 인식
     (문자열 안의 #, 이스케이프 따옴표 처리 / 삼중따옴표는 과정 코드에 없어 미처리)
   - 주석 부분만 `<span class="code-comment">`(#7ee787 녹색)로 감싼다
   - 복사 버튼은 하이라이팅과 무관하게 항상 원본 source를 복사
4. global.css 헤더 주석에 "accent는 오렌지 고정, 녹색은 code-comment 전용" 명시

## 재발 방지

외부 하이라이터(prism 등) 도입하지 말 것 — 주석 녹색이면 충분하다는 것이 확정 요구사항.
