# 10. 일자별 시간표 홈 + 좌측 책갈피 사이드바 (2026-07-17)

## 대표 지시

- 첫 페이지는 일자별 시간표
- 각 페이지에 시간별/책갈피용 좁은 왼쪽 메뉴

## 구현

### 홈 = 일자별 시간표
- 기존 3일 카드 그리드 → **DAY별 시간표 리스트**(.tt-day/.tt-row)로 교체
- 각 행: 시간 · n교시 · 제목 · 이론/실습 뱃지 · 완료 ✔ — 행 전체가 해당 교시로 링크
- DAY 헤더에 날짜·테마·진행률(n/7교시) 표시, DAY 라벨은 일차 페이지로 링크

### 공용 사이드바 (components/SideNav.jsx)
- 항목 타입 2종: `to`(라우트 이동) / `anchor`(scrollIntoView 책갈피)
  - **HashRouter라 URL 해시 앵커(#id)를 쓸 수 없음** → 버튼 + scrollIntoView 방식 (이유 주석 있음)
- 전역 `[id]{scroll-margin-top:80px}`로 스티키 헤더에 안 가리게 처리
- 데스크톱: 190px 스티키 / **모바일(≤860px): 상단 가로 스크롤 칩으로 전환** (.side-nav 미디어쿼리)

### 페이지별 적용
| 페이지 | 사이드바 내용 |
|--------|--------------|
| 일차(/day/:id) · 교시(/day/:id/:slot) | DaySideNav — DAY 1/2/3 전환 칩 + 해당 일차 7교시(시간·완료✔·현재 교시 하이라이트) |
| 선수학습 | 5개 영역 (필수/권장/준비 표시) |
| 실습소스 | DAY 헤더(굵게) + 코드 있는 교시 16개 (파일명 표시) |
| 프로젝트 | #1~#8 (난이도 표시) |
| About | 개발취지/강사소개/교육개요 |
| 홈·복습퀴즈·심화학습 | 사이드바 없음 (홈=시간표 자체가 내비, 퀴즈=선형 진행, 심화=짧음) |

### 기타
- 전역 규칙에 따라 `public/CNAME` 추가 (루트 CNAME과 양쪽 유지)
