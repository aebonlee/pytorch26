# 14. 미니 프로젝트 완성 소스 토글 (2026-07-17)

## 대표 지시

프로젝트를 제시만 하지 말고, 완성된 소스보기를 박스 오른쪽 아래에 배치해 열고 닫는 형식으로 소스도 제공.

## 구현

- projects.js 8개 전부에 `solution: { filename, source }` 추가 (sol01~sol08)
- 카드 우하단 "📄 완성 소스 보기 ▼ / 닫기 ▲" 토글 버튼(.sol-btn) — **기본 닫힘** (스스로 먼저 풀어보게)
- 열면 CodeBlock으로 표시 (복사 버튼·주석 녹색 하이라이팅 동일 적용)

- solution.source는 JS 템플릿 리터럴 — **백틱·달러중괄호 금지** (파이썬 f-string의 중괄호는 안전)
- #5·#6은 의도적으로 일부를 수강생 과제로 남김 (수업 코드 재사용 학습 목적) — "완성 안 됐다" 오해 없게 주석으로 명시되어 있음
