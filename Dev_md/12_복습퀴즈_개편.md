# 12. 복습퀴즈 개편 — 문항 사이드바 · 3회 응시 · 지문 수정 (2026-07-17)

## 대표 지시 3건

1. 복습퀴즈에도 왼쪽 메뉴 + 답안 선택 상태 표시
2. 채점결과를 3회 응시로 체크
3. "1번 문제만 '정답'이란 글자가 보기에 표시" 확인

## 구현

### 문항 사이드바
- Q1~Q15 책갈피 (질문 앞 22자 미리보기, 앵커 스크롤)
- 제출 전: 각 문항의 선택 답안(선택 ①~④ / 미선택) 표시, 타이틀에 응답 n/15
- 제출 후: 정답 ✔ / 오답(내 답 ①~④) 표시

### 3회 응시 제한 + 회차별 기록
- localStorage `pytorch26_quiz_attempts` = [{score, total, at(일시)}]
- 제출할 때마다 회차 기록 → 상단 "회차별 채점결과" 박스에 1~3회차 점수·일시 표시
- 3회 소진 시: 제출·다시풀기 비활성("3회 응시를 모두 마쳤습니다"), 마지막 채점 화면·해설은 열람 가능
- 초기화가 필요하면 브라우저 localStorage에서 해당 키 삭제 (관리 기능은 따로 없음 — 무기명 셀프 체크용)

### "정답" 글자 노출 확인 결과
- 원인: **Q1의 1번 보기 지문 자체**에 "정답 레이블 대신…"이라는 단어가 들어 있었음 (렌더링 버그 아님 — 다른 문항엔 없음, grep으로 확인)
- 조치: "미리 주어진 레이블 없이, 행동의 결과로 주어지는 평가적 보상만으로 학습한다"로 지문 수정
- **재발 방지: 보기 지문에 "정답"이라는 단어를 쓰지 말 것** (QuizPage 헤더 주석에도 기록)
