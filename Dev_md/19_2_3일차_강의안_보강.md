# 19. 2·3일차 강의안 대폭 보강 (2026-07-18)

18(1일차 보강)과 같은 원칙으로 2·3일차 전체 확장.
이론 블록 최종 집계(노드 검증): **1일차 38 / 2일차 28 / 3일차 30** (보강 전 각 17/17/14).

## 2일차 추가 내용

- **DQN 소개(5블록)**: 함수 근사=일반화 관점, 네트워크 입출력 설계(상태→모든 Q, 이산 전용인 이유), catastrophic forgetting, deadly triad 상세, 재현버퍼 3효과+off-policy 전제, 타깃넷 주기 트레이드오프·소프트 업데이트 예고
- **DDQN(4블록)**: 과대평가 전파·누적 메커니즘, van Hasselt 실측, Double Q-Learning 원조와의 관계, Dueling 상세(식별성 문제·V 공유 직관), Prioritized(IS 보정)·Noisy·Rainbow(절제 연구)
- **PyTorch(5블록)**: 배치 차원·shape 감각(gather 해설, (B,)vs(B,1) 브로드캐스팅 사고), autograd 상세(no_grad/detach의 RL적 의미), RL 목표=자가 생성 목표라는 관점, 시드·디바이스·item() 실무 팁
- **DQN 구현(4블록)**: CartPole 스펙(해결 기준 475), term/trunc 구분(시간초과≠종료), Huber 이유, 하이퍼 가이드, 증상별 처방(붕괴·정체)
- **PG(4블록)**: 로그-미분 트릭 전체 유도(환경 항 소거의 결정적 순간), cross-entropy와의 대응, causality(reward-to-go), 베이스라인 불변성 증명 직관, GAE·TRPO/PPO·RLHF 계보
- **AC(3블록)**: 역할 분담의 bias-variance 재해석, detach 필수 이유, 엔트로피 보너스↔SAC 복선, 공유 몸통 vs 분리, A3C/A2C/GAE 계보
- **A2C 구현(3블록)**: n-step 목표 계산(done 마스크 버그), gradient clipping 필수성, 손실 계수 실험, on-policy 곡선 특성

## 3일차 추가 내용

- **DDPG 소개(5블록)**: 이산화·수치최적화가 안 되는 이유, DPG 연쇄법칙 읽는 법+확률적 PG와의 극한 관계, 소프트 업데이트가 하드 카피보다 나은 이유, OU vs 가우시안 노이즈, 한계→TD3 3처방
- **DDPG 구현(3블록)**: Pendulum 스펙(cos/sin 상태 표현 이유·보상식), 업데이트 순서·Actor 손실 부호·옵티마이저 분리 안전장치·lr 비대칭 이유, 성능 붕괴 증상별 처방
- **MaxEnt(5블록)**: 기존 탐험 장치=임시방편 회고, 엔트로피 복습(A2C 보너스와의 격차), soft V=log-sum-exp(soft의 어원), control as inference·볼츠만 물리 대응·로버스트 제어, soft Q-Learning→SAC 계보
- **SAC(6블록)**: 3요소를 각각 독립 블록으로 — tanh 보정항(누락=대표 버그), 트윈 Q=비관주의(연속공간에서 더 필요한 이유), 자동 α=라그랑주 승수(α 궤적 해석), rsample vs REINFORCE-식 분산 비교, 손실 3종 요약표
- **SAC 구현(4블록)**: 업데이트 순서·기울기 분리(a' 재샘플, α detach), lr 통일의 의미, 디버깅 4항(보정 누락→α 폭주 연쇄)
- **TAC 소개(4블록)**: 샤논 −log 발산=확률 0 금지 조항이라는 원인 규명, q-로그 유한성→행동 포기 가능, sparsemax·지지집합 축소, q 선택 가이드
- **TAC 구현(3블록)**: 수정 2지점·clamp 이유, 실험 프로토콜(회귀 테스트 문화), **3일 종합 계보 한 장 + 관통 3질문**(목표/탐험/안정화)
